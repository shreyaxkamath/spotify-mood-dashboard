An interactive data visualization dashboard built with React and Recharts that explores how emotion, genre, and popularity intersect across thousands of Spotify tracks.

## Objectives
- Explore whether popular music tends to sound "happier"
- Compare emotional profiles across genres
- Enable user-driven storytelling using filters and interactive charts

## Tech Stack
- React + Vite
- Recharts
- Python preprocessing for feature engineering
- Deployed via Vercel

## Features
- Dynamic filtering by genre, mood quadrant, and popularity
- Multi-series line chart showing mood trends by popularity
- Interactive scatter plot linking directly to Spotify tracks
- Per-genre comparative mood analysis
- Elegant pastel UI optimized for readability
